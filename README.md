# DigitalPiggyBank
Traditional savings accounts are often too passive and boring to motivate young customers to
save money regularly. To solve this, we developed the "Digital Piggy Bank System." This is
a goal-oriented savings prototype that uses simple gamification to make saving fun and
engaging. Instead of just holding money like a standard bank account, our system acts as a
smart financial friend who will help us save money. It asks for the user's monthly income and
expenses to calculate a safe savings limit. Based on this data, it estimates exactly when the
user will reach their goal, helping them save for their dreams without ruining their daily
budget.
